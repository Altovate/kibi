Connect to an external datasource

Here, you can connect to two types of external datasource, JDBC and REST:

JDBC datasources:

Siren Investigate, by default, displays data which is contained in its predefined Siren Elasticsearch backend. Configure a JDBC datasource to "federate" other backends (e.g. RDBMS and more). Once a JDBC datasource has been configured, one can map remote tables and views to "virtual indexes" using the Indexes and Relations screen.

REST datasources:

Configure a REST datasource to invoke a remote REST service. Typical use is via a REST query which is then visualized in a template displayed using a query viewer dashboard component.
