Entity URI Schemas

There are 2 different uri schemas for passing parameters to parametric queries:

  • sql://@TABLE@/@PKVALUE@ for sql queries
    Example: sql://people/123
  • @URI@ for sparql queries
    Example: http://example.com/people/123

Query is parametric when it contains one of the following placeholders:

  • @TABLE@ or @PKVALUE@ for sql queries
  • @URI@ for sparql queries
